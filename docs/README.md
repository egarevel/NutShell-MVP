## Creating Assets

### For GIFs:
- Use a screen recorder like [ScreenToGif](https://www.screentogif.com/) or [LICEcap](https://www.cockos.com/licecap/)
- Keep file size under 10MB
- Recommended dimensions: 1280x720 or 1920x1080
- Frame rate: 15-30 fps

### For Screenshots:
- Use high-resolution (at least 1920x1080)
- Save as PNG for crisp text
- Annotate with arrows/highlights if needed

### For Diagrams:
- Use [Excalidraw](https://excalidraw.com/) or [draw.io](https://draw.io)
- Export as PNG with transparent background
- Include clear labels and flow arrows
